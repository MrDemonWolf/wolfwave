## How to add a component

1. Build the SwiftUI view under `Views/Shared/` or `Views/Onboarding/Components/`.
2. Reference `DSColor` / `DSFont` / `DSSpace` / `DSRadius` tokens — no hardcoded numbers.
3. Add a markdown entry in `design-system/components/<name>.md` following the [template](https://github.com/mrdemonwolf/wolfwave/blob/main/design-system/components/README.md).
4. Add a row to the table above.
