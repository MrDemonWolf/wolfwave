## !song Commands
**TL;DR: Show your current track.**

These commands will reply to the user with the song you're playing right now.

| Command        | Description             |
| -------------- | ----------------------- |
| `!song`        | Now Playing info        |
| `!currentsong` | Now Playing info        |
| `!nowplaying`  | Now Playing info        |

---

## !last Commands
**TL;DR: Show your previous track.**

These commands will reply with the last track you finished playing.

| Command        | Description             |
| -------------- | ----------------------- |
| `!lastsong`    | Previously Played info  |
| `!last`        | Previously Played info  |
| `!prevsong`    | Previously Played info  |

---

## 🛑 Cooldowns & Spam
**TL;DR: No spamming allowed.**

You can set **Global** and **Per-User** cooldowns in Settings. This prevents people from spamming commands in chat.

- **Global Cooldown** (default 15s): How long everyone has to wait between commands.
- **User Cooldown** (default 15s): How long one specific person has to wait after they use a command.

<Callout type="info">
  **Mod Bypass**: Mods and the channel owner always bypass cooldowns. You can use commands as much as you like!
</Callout>
