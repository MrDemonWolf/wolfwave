<span className="docs-eyebrow">Guide</span>

Set how WolfWave looks. The choice covers the whole app, including the menu bar menu.

## Pick an appearance

Open **Settings → General** and choose one:

- **System.** Follows macOS, including the automatic light and dark schedule. This is the default.
- **Light.** Forces the light look everywhere, even if macOS is in dark mode.
- **Dark.** Forces the dark look everywhere, even if macOS is in light mode.

The picker shows System Settings-style preview thumbnails so you can see each option before you pick.

## What it affects

Your choice applies to the whole app: the Settings window, the now-playing card, every section, and the menu bar menu. There is nothing else to set.

Settings cards sit on a solid macOS surface, so they read cleanly in both light and dark without glowing too bright or washing out.

## Good to know

- **System** is the right pick if you use the macOS light/dark schedule and want WolfWave to match.
- Appearance is a preference, so it travels with [Back Up & Restore](/docs/backup).
