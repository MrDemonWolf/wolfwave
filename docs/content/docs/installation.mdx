Get WolfWave running on your Mac in minutes.

## Download

Download the latest DMG from the [GitHub Releases](https://github.com/MrDemonWolf/WolfWave/releases) page.

<Callout type="info">WolfWave requires **macOS 15.0 or later**.</Callout>

## Install

1. Open the downloaded `.dmg` file
2. Drag **WolfWave** to your **Applications** folder
3. Eject the DMG

## First Launch

1. Open WolfWave from your Applications folder
2. The onboarding wizard will guide you through initial setup
3. WolfWave will appear in your menu bar once setup is complete

<Callout type="info">
  WolfWave is signed and notarized by Apple — no Gatekeeper warnings on first launch.
</Callout>

## Onboarding Wizard

On first launch, WolfWave walks you through a 3-step guided setup:

### Step 1: Welcome

An overview of WolfWave's features:
- Music Monitoring
- Twitch Chat Bot
- Discord Rich Presence
- Stream Overlays
- Menu Bar integration

When prompted by macOS, click **OK** to allow Apple Music access (required for tracking).

### Step 2: Connect Twitch (Optional)

To enable chat commands like `!song`:

1. Click **Connect with Twitch** in the onboarding wizard
2. Follow the device authorization flow in your browser
3. Return to WolfWave — you're connected!

You can skip this step and connect later in Settings.

### Step 3: Discord Rich Presence (Optional)

A visual preview shows what your Discord profile will look like with Rich Presence active.

1. Toggle **Enable Rich Presence** on to activate
2. Make sure Discord is running on your Mac
3. WolfWave will connect automatically

<Callout type="info">
  You'll need a Discord Application ID from the
  [Discord Developer Portal](https://discord.com/developers/applications). See the
  [Usage Guide](/docs/usage#discord-rich-presence) for setup details.
</Callout>

You can skip this step and enable later in Settings.

## You're Ready!

WolfWave is now tracking your Apple Music playback. Your viewers can use:

- `!song` — Shows the current track
- `!currentsong` — Shows the current track
- `!nowplaying` — Shows the current track
- `!lastsong` — Shows the previous track
- `!last` — Shows the previous track
- `!prevsong` — Shows the previous track
