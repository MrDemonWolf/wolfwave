WolfWave is a native macOS menu bar app that bridges Apple Music with your stream. It broadcasts the currently playing song to Twitch chat, shows "Listening to Apple Music" on Discord, and streams now-playing data to OBS overlays — all with zero external dependencies.

## Features

- **Real-time Now Playing**: Instant track detection via ScriptingBridge — no polling hacks.
- **Twitch Chat Bot**: Responds to `!song`, `!lastsong`, and more via the modern EventSub + Helix API (no deprecated IRC).
- **Discord Rich Presence**: "Listening to Apple Music" status with dynamic album art, elapsed time, and track details.
- **OBS Stream Overlay**: Built-in WebSocket server + bundled widget served at `http://localhost:8766` — add it as a Browser Source in OBS with no external dependencies.
- **Secure Credentials**: All tokens stored in macOS Keychain — never in plain text.
- **Guided Onboarding**: First-launch wizard walks you through Twitch, Discord, and OBS setup in minutes.
